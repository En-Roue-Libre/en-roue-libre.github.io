Nos addons

Raiduku est un addon minimaliste pour aider les raid leads à gérer les loots dans WotLK Classic :

  • Tous les raiders n'ont pas besoin de l'installer. Seulement celui qui gère les loots.
  • Pas intrusif. Vous pouvez toujours attribuer le butin manuellement.
  • Supporte le système +1, ordonne les rands et désigne automatiquement le gagnant.
  • Supporte le rand inversé (1 > 100).
  • Supporte thatsmybis.com permettant d'importer les prios et d'exporter l'historique des loots (même s'ils sont attribués manuellement).
  • Supporte softres.it.


AutoLoggerWotlk est un addon qui active automatiquement les logs de combat avancés dans tous les raids WotLK actuels et à venir.

Addons favoris

  • Weak Auras : l' addon de tous les addons. Une fois l'addon installé, ça se passe ici: wago.io.
  • GTFO : le feu ca brûle et si tu ne le vois pas, maintenant tu l'entends.
  • DBM ou Big Wigs : la base pour tout raideur.
  • Details! : avant tout pour comprendre comment se passent les combats, mais surtout utilisé comme kiki meter.
  • OmniCC : parce qu'avoir un vrai décompte sur les CDs c'est mieux.
  • Leatrix Plus : plein d'options qui améliorent la qualité de vie.
  • Leatrix Maps : la map mais en mieux.
