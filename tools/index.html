Nos addons

Raiduku est un addon minimaliste pour aider les raid leads grer les loots dans WotLK Classic :

  • Tous les raiders n'ont pas besoin de l'installer. Seulement celui qui gre les loots.
  • Pas intrusif. Vous pouvez toujours attribuer le butin manuellement.
  • Supporte le systme +1, ordonne les rands et dsigne automatiquement le gagnant.
  • Supporte le rand invers (1 > 100).
  • Supporte thatsmybis.com permettant d'importer les prios et d'exporter l'historique des loots (mme s'ils sont attribus manuellement).
  • Supporte softres.it.


AutoLoggerWotlk est un addon qui active automatiquement les logs de combat avancs dans tous les raids WotLK actuels et venir.
